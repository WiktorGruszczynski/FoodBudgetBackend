# FoodBudget - backend

### Features:
* Calories and macros [Protein, Fats, Carbohydrates]
* Prices of prepared meals 

---

### Project setup

1. **Install python virtual env**

   ```bash
   python -m venv .venv
   source .venv/bin/activate        # Linux / macOS
   .venv\Scripts\activate           # Windows
   ```

2. **Install requirements and dev requirements**

   ```bash
   pip install -r requirements.txt
   
   ```

---

### Dev tools setup

1. **Install dev tools**

   ```bash
      pip install -r requirements-dev.txt
   ```

2. **Install pre-commit hooks**

   ```bash
      pre-commit install
   ```
